ABOUT THE APP

Key features

Networking

Create a network of other AP students, all helping each other succeed.

Saves Time

Share notes amongst all AP students, saving your individual time used to take notes

Reliable

User reviews help make sure that each user is held accountable for what they post.

Customizable

Customizable folders and notes help make study plans unique to each user.

Groundbreaking

There is no other single app or website that allows users to compile AP class notes and share them in a network for free.

FINAL

What we hope the final product will look like

  • User Ratings
  • Each user can rate notes uploaded by others.
  • Ratings will show up on the user's profile.
  • Users with higher ratings have better or neater notes.
  • See what other's think of the user or of the notes.
  • Customization
  • Create and Edit folders for a unique user experience.
  • Choose your favorite notes to save.
  • Edit your profile accordingly.
  • Create the app according to which AP classes you take.
  • Upload
  • User can upload from Camera Roll or take a picture.
  • The user names their notes and saves it into an AP subject category.
  • User must upload one note to access 5 notes.
  • Helps to center the app around user contributions
